## MINIKUBE PRACTICAL IMPLEMENTATION

Step 1: Install prerequisites
Enable virtualization in BIOS
Install any one: Hyper-V or Docker Desktop or VirtualBox

Step 2: Download and install Minikube
Download installer from
[https://storage.googleapis.com/minikube/releases/latest/minikube-installer.exe](https://storage.googleapis.com/minikube/releases/latest/minikube-installer.exe)
Run the installer normally

Step 3: Add Minikube to PATH if required
Add the folder
C:\Program Files\Minikube
to system PATH

Step 4: Start Minikube
minikube start --driver=docker

Check status
minikube status

Step 5: Use kubectl
List pods
minikube kubectl -- get pods -A

Open dashboard
minikube dashboard

## DEPLOYING APPLICATION ON MINIKUBE

Step 1: Create deployment
kubectl create deployment mynginx --image=nginx

Check deployment
kubectl get deployments
kubectl get pods
kubectl describe pods

Step 2: Expose deployment
kubectl expose deployment mynginx --type=NodePort --port=80 --target-port=80

Step 3: Scale application
kubectl scale deployment mynginx --replicas=4

Step 4: Access application

Port forward
kubectl port-forward svc/mynginx 8081:80
Open browser:
[http://localhost:8081](http://localhost:8081)

Or using tunnel
minikube tunnel
minikube service mynginx --url

View dashboard
minikube dashboard

Step 5: Cleanup
kubectl delete deployment mynginx
kubectl delete service mynginx
minikube stop
minikube delete

## NAGIOS MONITORING PRACTICAL

Step 1: Pull image
docker pull jasonrivers/nagios:latest

Step 2: Run Nagios
docker run --name nagiosdemo -p 8888:80 jasonrivers/nagios:latest

Step 3: Access dashboard
[http://localhost:8888](http://localhost:8888)
Username: nagiosadmin
Password: nagios

Step 4: Stop and remove
docker stop nagiosdemo
docker rm nagiosdemo

Optional remove image
docker rmi jasonrivers/nagios:latest
